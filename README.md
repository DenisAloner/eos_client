eos_client  
==========  
Сборка проекта:  
1. Создать папку "..\eos_client\build"  
2. Перейти в данную папку (cd <путь к папке ..\eos_client\build>)  
3. Выполнить "..\cmake_build\win\gen 14"  
4. Запускать проект "eos_client"
