[Расчет урона]

Для оружия ближнего боя:
 STR - сила персонажа;
 MOW - мастерство владения оружием данного типа (%);
 DWM - требование к мастерству;
 DMG - конечный урон;
 BDW - базовый урон оружия;
 BOS - бонус силы (%);
 LP - освещенность(%)
 Accuracy - точность;

 Accuracy = (MOW + RND(MOW-DWM)) / 100 * (LP + RND(100-LP)/2 ) / 100;
 DMG = Accuracy*BDW*STR*BOS/100;

Типы оружия ближнего боя:
 - мечи
 - клинки
